# FitManager 💪

**Sistema de gestão para Personal Trainers** — Trabalho final de Marketing Digital (ISULPAR · Sistemas de Informação).
Categoria sorteada: **SaaS**.

## O que tem neste projeto

| Arquivo | O que é |
|---|---|
| `index.html` | **Landing page** de divulgação (marketing, SEO, conversão) |
| `app.html` | **Aplicativo SaaS** funcional (gestão de alunos, treinos e pagamentos) |
| `assets/` | CSS e JavaScript do site e do app |
| `RELATORIO.docx` / `RELATORIO.md` | **Relatório prático** completo para entrega |

## Como rodar (não precisa instalar nada)

1. Dê **duplo clique em `index.html`** para abrir a landing page no navegador.
2. Clique em **"Testar grátis"** ou **"Entrar"** para abrir o app (`app.html`).
3. No login, use os dados já preenchidos:
   - **E-mail:** `personal@fitmanager.com`
   - **Senha:** `123456`

> Os dados (alunos, treinos, pagamentos) ficam salvos no próprio navegador (localStorage).
> Para ver os vídeos dos exercícios é necessário internet (vídeos do YouTube).

## Funcionalidades do app

- **Painel:** receita do mês, alunos ativos, inadimplência e gráfico de receita.
- **Alunos:** cadastro, busca, ficha com avaliação física, treino e histórico de pagamentos.
- **Treinos:** montar fichas com séries/repetições/descanso e atribuir a alunos.
- **Exercícios:** biblioteca com fotos e vídeos demonstrativos.
- **Pagamentos:** controle de mensalidades, marcar como pago e registrar cobranças.
